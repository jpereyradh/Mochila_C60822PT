a. ¿Quién inventó el sistema de control de versión Git y por qué?
Linus Torvalds, el mismísimo creador de Linux, desarrolló e implementó la primera versión de Git
para dar a sus programadores del kernel una alternativa a BitKeeper. Linus no sólo hizo el diseño
original sino que fue vital para promover el proyecto con su estilo único.

c. ¿Hay otra forma que no sea la terminal para trabajar con Github?
Se puede utilizar GitHub Desktop para crear y administrar un repositorio de Git sin utilizar 
la línea de comandos. Usando una interfaz visual en lugar de comandos de texto en la línea de comandos para simplificar el flujo 
de trabajo de GitHub.
